## React in Practice

### Kenapa Kita Perlu Men-struktur Project dengan Baik di React?

1. Supaya mudah di-maintain (mudah dipelihara, diatur).
2. Mempercepat detect masalah.
3. Supaya tidak pusing bila project semakin besar.
